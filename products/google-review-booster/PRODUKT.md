## 1. Das Problem

Handwerker wissen, dass Google-Bewertungen wichtig sind — aber:
- Sie vergessen, Kunden danach zu fragen
- Es ist ihnen unangenehm ("Können Sie mir eine Bewertung schreiben?")
- Selbst wenn sie fragen: der Kunde vergisst es
- Nur unzufriedene Kunden schreiben von sich aus Bewertungen

**Ergebnis:** 3-Sterne-Profil mit 8 Bewertungen — während die Konkurrenz 4,8 Sterne mit 120 Bewertungen hat.

---

## 2. Die Lösung

Ein automatisierter Bewertungs-Funnel in 3 Schritten:

```
Auftrag fertig
     │
     ▼
┌────────────────────────────────┐
│ Handwerker öffnet App/Link    │
│ → Tippt Kundennamen ein       │
│ → Klickt "Link senden"        │
└────────────┬───────────────────┘
             │  WhatsApp / SMS / E-Mail
             ▼
┌────────────────────────────────┐
│ Kunde bekommt freundliche      │
│ Nachricht:                     │
│ "Hat alles gepasst? Wir       │
│  freuen uns über Ihr Feedback" │
│                                │
│    [★★★★★ Ja, super!]         │
│    [★★★ Geht so]              │
└────────────┬───────────────────┘
             │
     ┌───────┴───────┐
     ▼               ▼
  4-5 Sterne      1-3 Sterne
     │               │
     ▼               ▼
  → Google        → Privates
    Review          Feedback-
    Seite           Formular
    (direkt)        (intern)
```

**Der Trick:** Nur zufriedene Kunden werden zu Google weitergeleitet. Unzufriedene geben privat Feedback → der Handwerker kann reagieren, BEVOR es eine schlechte Bewertung gibt.

## 4. Warum €29/Monat?

### ROI-Rechnung für den Handwerker:
```
Durchschnittlicher Auftragswert:        €2.500
Neukunden durch Google pro Monat:        +2 (konservativ)
Zusätzlicher Umsatz:                     €5.000/Monat
Kosten Review Booster:                   €29/Monat
ROI:                                     17.000%
```

**Verkaufsargument:** *"Für weniger als einen Kaffee am Tag bekommen Sie systematisch mehr Google-Bewertungen. Eine einzige Bewertung, die einen Neukunden bringt, hat sich schon 100-fach bezahlt gemacht."*
