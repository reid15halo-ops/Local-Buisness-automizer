## Action Buttons (Use verbs!)

| Bad | Good |
|---|---|
| Submit | Absenden |
| OK | Bestätigen |
| Yes | Ja, [action verb] |
| Confirm | [Repeat the action, e.g., "Ja, Rechnung senden"] |
| Process | Verarbeiten → Be specific: "Auftrag starten" |
| Generate | Erstellen |
| Sync | — (hide from user entirely) |
| Refresh | Aktualisieren |
| Retry | Nochmal versuchen |

## Error States

| Technical | Human German |
|---|---|
| Error 404 | "Diese Seite gibt es leider nicht." |
| Error 500 | "Es ist ein Fehler aufgetreten. Bitte versuchen Sie es später erneut." |
| Network error | "Keine Internetverbindung. Bitte prüfen Sie Ihre Verbindung." |
| Validation error | "Bitte prüfen Sie Ihre Eingaben." |
| Timeout | "Die Anfrage hat zu lange gedauert. Bitte versuchen Sie es erneut." |
| Unauthorized | "Bitte melden Sie sich erneut an." |
| Conflict | "Diese Änderung konnte nicht gespeichert werden, da jemand anderes die Daten gerade bearbeitet hat." |

## Empty States

Never show a blank screen. Always show a friendly message + action:

| View | Empty State Message | Action Button |
|---|---|---|
| Kunden | "Sie haben noch keine Kunden angelegt." | "Ersten Kunden anlegen" |
| Angebote | "Sie haben noch keine Angebote erstellt." | "Erstes Angebot schreiben" |
| Aufträge | "Noch keine Aufträge vorhanden." | "Neuen Auftrag anlegen" |
| Rechnungen | "Noch keine Rechnungen erstellt." | "Erste Rechnung schreiben" |
| Lager | "Ihr Lager ist noch leer." | "Erstes Material anlegen" |
